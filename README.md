# Boxing Performance Evaluation Platform

This project aims to provide a comprehensive performance evaluation platform for boxers using various technologies.

## Project Structure

```plaintext
Boxing-App/
│
├── backend/
│   ├── app/
│   ├── requirements.txt
│   └── main.py
│
├── frontend/
│   ├── lib/
│   ├── pubspec.yaml
│   └── main.dart
│
├── database/
│   └── schema.sql
│
└── ai_module/
    └── model.py
```

## Description
- **backend/**: Contains the FastAPI application files.
- **frontend/**: Contains the Flutter application files.
- **database/**: Contains the PostgreSQL database schema.
- **ai_module/**: Contains the AI model files for performance evaluation.
